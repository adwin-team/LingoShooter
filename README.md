# LingoShooter - 語学学習シューティングゲーム

LingoShooter は、流れてくる英語の問題に素早く回答して敵を撃破する、ブラウザベースの語学学習シューティングゲームです。

## 🎮 ゲーム概要

敵が画面奥から迫ってきます。提示される 4 択問題に正解することで攻撃を行い、敵を倒すことができます。回答に時間がかかったり、誤答したりするとダメージを受け、HP が 0 になるとミッション失敗です。

### 特徴
- **プレミアムなビジュアル**: ダークテーマとネオンカラーを採用した近未来的な UI。
- **動的な難易度**: 正解を重ねるごとに敵の接近スピードが上昇します。
- **学習の最適化**: 
  - 選択肢は毎回ランダムにシャッフルされます。
  - 同じ問題が連続して出題されないように制御されています。
- **豊富な問題**: 日常会話、ビジネス、旅行、英文法など 20 問以上を収録。

## 🚀 はじめ方

1. このリポジトリをダウンロードまたはクローンします。
2. `index.html` を任意のモダンブラウザ（Chrome, Firefox, Edge 等）で開きます。
3. 「START MISSION」をクリックしてゲームを開始します。

> [!NOTE]
> 通常の `file://` プロトコルで開いた場合でも動作するようにフォールバックが実装されていますが、`questions.json` からの動的な読み込みを完全に体験するにはローカルサーバー（Python の `http.server` 等）での実行を推奨します。

## 🛠 技術スタック

- **Frontend**: HTML5, Vanilla CSS, JavaScript (ES6+)
- **Data**: JSON
- **Design Concepts**: Glassmorphism, Neon Aesthetics

## フォルダ構成

- `index.html`: ゲームのメイン画面
- `style.css`: パーティクル風のエフェクトやモダンなスタイリング
- `script.js`: ゲームエンジン、問題制御ロジック
- `questions.json`: 問題データ（JSON形式で拡張可能）
